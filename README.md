# market-basket-analysis-ml
# 🛒 Market Basket Analysis using Machine Learning

## 📌 Overview

This project analyzes customer purchasing behavior using association rule mining to identify relationships between products to optimize operations in a local supermarket.

## 🎯 Objective

To discover product combinations frequently bought together and generate actionable business insights.

## 📊 Dataset

* Retail transaction dataset, customer data and products data.

## 🛠️ Tools & Technologies

* Python
* Pandas
* Mlxtend (Apriori Algorithm)
* Sklearn
* Pycharm
* Streamlit

## ⚙️ Methodology

1. Data preprocessing
2. Transaction encoding
3. Apriori algorithm implementation.
4. Association rule generation.
5. Evaluation using support, confidence, and lift.
6. Training predictive models.(K-means, Random Forest classifier)

## 📈 Results

* Identified strong product associations.
* Generated rules for cross-selling strategies.
* Generated actionable business recommendations.
* Improved understanding of customer behavior.
